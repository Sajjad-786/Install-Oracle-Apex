# Portainer Dashboard – Screenshot Notes

This folder contains screenshots of the Portainer installation on the **Dev APEX Server**.

## What to capture here

- Initial Portainer login screen
- Dashboard overview after first login
- Container list showing Oracle DB, ORDS, and NGINX containers
- Stack deployment screen
- Volume and network overview

## Naming convention

Please name screenshots in the following format:

```
YYYY-MM-DD_portainer_<description>.png
```

**Example:** `2026-05-14_portainer_dashboard-overview.png`
